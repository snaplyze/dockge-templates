# EdgeConnector 840D

Access Siemens SINUMERIK 840D sl/pl controllers and provide data via OPC UA and MQTT

## Quick Start

```bash
cd /opt/stacks/edgeconnector840d
docker compose up -d
```

## Ports

- `443`: Service port
- `8099`: Service port
- `4897`: Service port
