# Frigate NVR

A complete and local NVR designed for Home Assistant with AI object detection. Uses OpenCV and Tensorflow to perform realtime object detection locally for IP cameras.

## Quick Start

```bash
cd /opt/stacks/frigatenvr
docker compose up -d
```
