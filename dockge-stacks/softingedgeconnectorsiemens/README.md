# Softing EdgeConnector Siemens

Connect Siemens SIMATIC S7-300/400/1200/1500 PLCs and provide the data via OPC UA and MQTT

## Quick Start

```bash
cd /opt/stacks/softingedgeconnectorsiemens
docker compose up -d
```
