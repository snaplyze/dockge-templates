# Plone

A free and open-source CMS built on top of Zope

## Important Note

Default user and password are admin/admin

## Quick Start

```bash
cd /opt/stacks/plone
docker compose up -d
```

## Ports

- `8080`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/data` in container
