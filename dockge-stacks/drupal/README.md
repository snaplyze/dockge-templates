# Drupal

Open-source content management framework

## Quick Start

```bash
cd /opt/stacks/drupal
docker compose up -d
```

## Ports

- `80`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/var/www/html` in container
