# IronFunctions UI

Open-source user interface for IronFunctions

## Quick Start

```bash
cd /opt/stacks/ironfunctionsui
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `API_URL`: API URL

## Ports

- `4000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/app/data` in container
