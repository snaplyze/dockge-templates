# Dashy

Helps you organize your self-hosted services by making them accessible from a single place.

## Quick Start

```bash
cd /opt/stacks/dashy
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `PUID`: PUID
- `PGID`: PGID
- `PORT`: PORT
