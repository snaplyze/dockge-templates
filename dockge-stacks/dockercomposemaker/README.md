# Docker Compose maker

DCM (Docker Compose Maker) is a simple yet powerful tool that helps you create docker-compose.yaml files for your self-hosted applications.

## Quick Start

```bash
cd /opt/stacks/dockercomposemaker
docker compose up -d
```
