# Redis

Open-source in-memory data structure store

## Quick Start

```bash
cd /opt/stacks/redis
docker compose up -d
```

## Ports

- `6379`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/data` in container
