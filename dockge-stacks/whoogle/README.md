# Whoogle

Self-hosted, ad-free, privacy-respecting Google metasearch engine.

## Quick Start

```bash
cd /opt/stacks/whoogle
docker compose up -d
```

## Ports

- `5001:5000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/config` in container
