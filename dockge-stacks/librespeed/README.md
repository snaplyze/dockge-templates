# LibreSpeed

A Free and Open Source Speedtest for HTML5 and more.

## Quick Start

```bash
cd /opt/stacks/librespeed
docker compose up -d
```

## Ports

- `80`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/config` in container
