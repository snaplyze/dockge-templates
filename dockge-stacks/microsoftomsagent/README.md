# Microsoft OMS Agent

Microsoft Operations Management Suite Linux agent.

## Quick Start

```bash
cd /opt/stacks/microsoftomsagent
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `AZURE_WORKSPACE_ID`: Workspace ID
- `AZURE_PRIMARY_KEY`: Primary key
