# Chowdown

Have recipes in seconds with plain text formatting and create beatiful recipe pages with automated ease.

## Quick Start

```bash
cd /opt/stacks/chowdown
docker compose up -d
```

## Ports

- `4000:4000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/config` in container
