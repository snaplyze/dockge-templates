# Authentik

Authentik is an open-source Identity Provider focused on flexibility and versatility

## Quick Start

```bash
cd /opt/stacks/authentik
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `PUID`: PUID
- `PGID`: PGID
- `PORT`: PORT
