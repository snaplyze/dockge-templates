## Quick Start

```bash
cd /opt/stacks/wg-easy
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `WG_HOST`: WG_HOST
- `PASSWORD`: PASSWORD
- `WG_PORT`: WG_PORT
- `WG_DEFAULT_DNS`: WG_DEFAULT_DNS
- `WG_DEFAULT_ADDRESS`: WG_DEFAULT_ADDRESS
- `WG_ALLOWED_IPS`: WG_ALLOWED_IPS
