# Postiz

 An alternative to: Buffer.com, Hypefury, Twitter Hunter, etc...Postiz offers everything you need to manage your social media posts,build an audience, capture leads, and grow your business.

## Quick Start

```bash
cd /opt/stacks/postiz
docker compose up -d
```
