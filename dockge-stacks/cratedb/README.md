# CrateDB

An open-source distributed SQL database

## Quick Start

```bash
cd /opt/stacks/cratedb
docker compose up -d
```

## Ports

- `4200`: Service port
- `4300`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/data` in container
