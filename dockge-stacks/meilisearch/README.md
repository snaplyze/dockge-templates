# Meilisearch

Meilisearch is a **RESTful search API**. It aims to be a **ready-to-go solution** for everyone who wants a **fast and relevant search experience** for their end-users

## Quick Start

```bash
cd /opt/stacks/meilisearch
docker compose up -d
```
