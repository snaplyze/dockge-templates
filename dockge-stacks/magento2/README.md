# Magento 2

Open-source e-commerce platform

## Quick Start

```bash
cd /opt/stacks/magento2
docker compose up -d
```

## Ports

- `80`: Service port
- `3000`: Service port
- `3001`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/var/www/html/` in container
