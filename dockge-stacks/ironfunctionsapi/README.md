# IronFunctions API

Open-source serverless computing platform

## Quick Start

```bash
cd /opt/stacks/ironfunctionsapi
docker compose up -d
```

## Ports

- `8080`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/app/data` in container
