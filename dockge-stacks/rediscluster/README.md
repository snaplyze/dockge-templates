# Redis Cluster

Open-source in-memory data structure store - Cluster mode

## Quick Start

```bash
cd /opt/stacks/rediscluster
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `REDIS_PASSWD`: Redis password
