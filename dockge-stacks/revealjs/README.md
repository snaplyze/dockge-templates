# Reveal.js

It is an open source HTML presentation framework. It enables anyone with a web browser to create fully featured and beautiful presentations for free | Production Mode

## Quick Start

```bash
cd /opt/stacks/revealjs
docker compose up -d
```

## Ports

- `6060:8080`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/reveal/docs/slides` in container
- `./data/*`: Mapped to `/resources` in container
