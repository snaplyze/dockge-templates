# Sshwifty

Sshwifty is a SSH and Telnet connector made for the Web. It can be deployed on your computer or server to provide SSH and Telnet access interface for any compatible (standard) web browser.

## Important Note

Template created by Pi-Hosted SeriesCheck our Github page: https://github.com/pi-hosted/pi-hostedOfficial Webpage: https://sshwifty.herokuapp.com/Official Docker Documentation: https://github.com/nirui/sshwifty#docker-image

## Quick Start

```bash
cd /opt/stacks/sshwifty
docker compose up -d
```
