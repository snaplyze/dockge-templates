# Petio

Petio is a third party companion app available to Plex server owners to allow their users to request, review and discover content.

## Quick Start

```bash
cd /opt/stacks/petio
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `PUID`: PUID
- `PGID`: PGID
- `TZ`: TZ
- `PORT`: PORT
