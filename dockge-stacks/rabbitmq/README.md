# RabbitMQ

Highly reliable enterprise messaging system

## Quick Start

```bash
cd /opt/stacks/rabbitmq
docker compose up -d
```

## Ports

- `5671`: Service port
- `5672`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/var/lib/rabbitmq` in container
