# NextcloudStack

Nextcloud setup with a MySQL database

## Important Note

Deploys a Nextcloud instance connected to a MySQL database.

## Quick Start

```bash
cd /opt/stacks/nextcloudstack
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `mypassword`: MYSQL_PASSWORD
- `myrpassword`: MYSQL_ROOT_PASSWORD
