# Nginx Proxy Manager

Nginx Proxy Manager enables you to easily forward to your websites running at home or otherwise, including free SSL, without having to know too much about Nginx or Letsencrypt.

## Quick Start

```bash
cd /opt/stacks/npm
docker compose up -d
```

## Ports

- `8000:80`: Service port
