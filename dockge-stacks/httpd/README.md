# Httpd

Open-source HTTP server

## Quick Start

```bash
cd /opt/stacks/httpd
docker compose up -d
```

## Ports

- `80`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/usr/local/apache2/htdocs/` in container
