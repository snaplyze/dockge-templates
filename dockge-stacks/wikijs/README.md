# Wikijs

Wikijs A modern, lightweight and powerful wiki app built on NodeJS.

## Quick Start

```bash
cd /opt/stacks/wikijs
docker compose up -d
```

## Ports

- `3100:3000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/config` in container
- `./data/*`: Mapped to `/data` in container
