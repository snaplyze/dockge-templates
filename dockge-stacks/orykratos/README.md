# Ory Kratos (Standalone)

Ory Kratos with Oathkeeper is _the_ developer-friendly, security-hardened and battle-test Identity, User Management and Authentication system for the Cloud.

## Quick Start

```bash
cd /opt/stacks/orykratos
docker compose up -d
```
