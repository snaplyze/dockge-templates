# Mongo

Open-source document-oriented database

## Quick Start

```bash
cd /opt/stacks/mongo
docker compose up -d
```

## Ports

- `27017`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/data/db` in container
