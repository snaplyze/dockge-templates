# Registry

Docker image registry

## Quick Start

```bash
cd /opt/stacks/registry
docker compose up -d
```

## Ports

- `5000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/var/lib/registry` in container
