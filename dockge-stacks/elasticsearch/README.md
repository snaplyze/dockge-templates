# Elasticsearch

Open-source search and analytics engine

## Quick Start

```bash
cd /opt/stacks/elasticsearch
docker compose up -d
```

## Ports

- `9200`: Service port
- `9300`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/usr/share/elasticsearch/data` in container
