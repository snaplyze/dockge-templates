## Quick Start

```bash
cd /opt/stacks/penpot-http
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `PUID`: PUID
- `PGID`: PGID
- `PENPOT_FLAGS_FE`: Frontend Flags
- `PENPOT_FLAGS_BE`: Backend Flags
- `PUBLIC_URI`: Public URI
- `TELEMETRY_ENABLED`: Penpot Telemetry Enabled?
- `PORT`: Http Port
