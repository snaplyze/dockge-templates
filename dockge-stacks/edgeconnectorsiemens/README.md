# EdgeConnector Siemens

Connect Siemens SIMATIC S7-300/400/1200/1500 PLCs and provide the data via OPC UA and MQTT

## Quick Start

```bash
cd /opt/stacks/edgeconnectorsiemens
docker compose up -d
```

## Ports

- `443`: Service port
- `8099`: Service port
- `4897`: Service port
