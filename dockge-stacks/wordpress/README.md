# WordPress

WordPress setup with a MySQL database

## Important Note

Deploys a WordPress instance connected to a MySQL database.

## Quick Start

```bash
cd /opt/stacks/wordpress
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `MYSQL_DATABASE_PASSWORD`: Database root password
