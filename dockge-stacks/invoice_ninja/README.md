# Invoice Ninja

Invoices, Expenses and Tasks built with Laravel and Flutter.

## Important Note

The database user is invoice_ninja and the database is ninja_db. Please generate an app key following the documentation here. 

## Quick Start

```bash
cd /opt/stacks/invoice_ninja
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `URL`: URL
- `APP_KEY`: APP_KEY
- `TZ`: TZ
- `DATABASE_PASSWORD`: DATABASE_PASSWORD
- `MYSQL_ROOT_PASSWORD`: MYSQL_ROOT_PASSWORD
- `PORT`: PORT
