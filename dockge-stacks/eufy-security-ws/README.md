# Eufy Security WS

Eufy Security Web Socket. Edit container and add username & password.

## Quick Start

```bash
cd /opt/stacks/eufy-security-ws
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `USERNAME`: USERNAME
- `PASSWORD`: PASSWORD

## Ports

- `3993:3000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/data` in container
