# Audiobookshelf

Self hosted audiobook and podcast server

## Quick Start

```bash
cd /opt/stacks/audiobookshelf
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `PUID`: PUID
- `PGID`: PGID
- `PORT`: PORT
