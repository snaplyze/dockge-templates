# Huginn

Create agents that monitor and act on your behalf.

## Quick Start

```bash
cd /opt/stacks/huginn
docker compose up -d
```

## Ports

- `3000:3000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/var/lib/mysql` in container
