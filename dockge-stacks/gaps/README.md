# Gaps

Gaps searches through your Plex Server or local folders for all movies, then queries for known movies in the same collection.

## Quick Start

```bash
cd /opt/stacks/gaps
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `TZ`: TZ

## Ports

- `8484`: Service port
