# TiddlyWiki

A unique, non-linear notebook wiki.

## Quick Start

```bash
cd /opt/stacks/tiddlywiki
docker compose up -d
```

## Ports

- `8080:8080`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/var/lib/tiddlywiki` in container
