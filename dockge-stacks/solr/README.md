# Solr

Open-source enterprise search platform

## Quick Start

```bash
cd /opt/stacks/solr
docker compose up -d
```

## Ports

- `8983`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/opt/solr/mydata` in container
