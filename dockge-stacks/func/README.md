# OpenFaaS

Serverless functions made simple

## Important Note

Deploys the API gateway and sample functions. You can access the UI on port 8080. Warning: the name of the stack must be 'func'.

## Quick Start

```bash
cd /opt/stacks/func
docker compose up -d
```
