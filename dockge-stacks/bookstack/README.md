# Bookstack

Bookstack is a free and open source Wiki designed for creating beautiful documentation. Feautring a simple, but powerful WYSIWYG editor it allows for teams to create detailed and useful documentation with ease.

## Important Note

Default login is admin@admin.com with a password of password. The database created is called bookstackapp and the database user is called bookstack

## Quick Start

```bash
cd /opt/stacks/bookstack
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `PUID`: PUID
- `PGID`: PGID
- `TZ`: TZ
- `DATABASE_PASSWORD`: DATABASE_PASSWORD
- `MYSQL_ROOT_PASSWORD`: MYSQL_ROOT_PASSWORD
- `PORT`: PORT
