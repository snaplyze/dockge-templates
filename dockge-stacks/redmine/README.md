# Redmine

Open-source project management tool

## Important Note

Default user and password are admin/admin

## Quick Start

```bash
cd /opt/stacks/redmine
docker compose up -d
```

## Ports

- `3000`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/usr/src/redmine/files` in container
