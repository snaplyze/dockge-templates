# Floating License Server

License Server for Softing edgeConnector products

## Quick Start

```bash
cd /opt/stacks/floatinglicenseserver
docker compose up -d
```

## Ports

- `6200`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/licsrv/licenses` in container
