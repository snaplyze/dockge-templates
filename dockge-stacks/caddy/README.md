# Caddy

Open-source web server with automatic HTTPS written in Go

## Quick Start

```bash
cd /opt/stacks/caddy
docker compose up -d
```

## Ports

- `80`: Service port

## Data Volumes

Your data is stored in:

- `./data/*`: Mapped to `/data` in container
