# LiveSwitch

A basic LiveSwitch stack with gateway, caching, database and media server

## Quick Start

```bash
cd /opt/stacks/liveswitch
docker compose up -d
```

## Configuration

Edit `.env` file to customize your deployment:

- `POSTGRES_PASSWORD`: Postgres password
